IoT & Non-IoT Classifier:
Distinguishing Network Traffic

This is my dissertation project which aimed to create five machine learning models that can distinguish between IoT and non-IoT network traffic and evaluate the five of 
them based on How accurate they are, how fast they are, and how they do on resource constrained machines.

The code runs off the Juypter notebook and needs the dataset in the zip file to run.
